# QA Testing Enterprise

## Apply When
- Prompt includes `#QC` or `#Test`.
- Task requires test planning, quality gates, regression coverage, or release sign-off criteria.

## Core Focus
- Define risk-based test strategy and acceptance criteria.
- Ensure deterministic, stable, and maintainable automated tests.
- Validate functional, integration, and non-functional quality requirements.

## Delivery Rules
- Prioritize critical business flows and failure scenarios.
- Track flaky tests and enforce reliability improvements.
- Require clear pass/fail gate definitions in CI/CD.
- Provide concise defect risk summaries and mitigation steps.
