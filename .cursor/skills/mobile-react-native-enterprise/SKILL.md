# Mobile React Native Enterprise

## Apply When
- Prompt includes `#MB`.
- Task concerns mobile architecture, screens, navigation, state, or release quality.

## Core Focus
- Build modular React Native architecture with feature boundaries.
- Enforce type-safe API contracts and predictable state flows.
- Optimize rendering, startup, and device performance.
- Keep code maintainable for long-lived enterprise products.

## Delivery Rules
- Separate UI, business logic, and data access layers.
- Reuse shared components and hooks; avoid duplication.
- Handle offline/network error states explicitly.
- Include testing strategy and release readiness checks.
